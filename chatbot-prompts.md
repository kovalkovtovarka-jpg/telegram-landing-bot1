# Промпты для чат-бота генератора лендингов

## Базовая структура промпта

```
Ты - эксперт по созданию лендингов для перепродажи товаров. 

Создай лендинг на основе следующей информации:
[ДАННЫЕ О ТОВАРЕ]

Используй шаблон: [ID_ШАБЛОНА]

Требования:
- Адаптивный дизайн для мобильных и десктопов
- HTML5, CSS3, vanilla JavaScript
- Интеграция с Telegram для отправки заказов
- TikTok Pixel для отслеживания
- Оптимизация для конверсии
```

---

## Промпты по категориям

### 1. Физический товар - Прямая продажа

```
Создай лендинг для продажи товара: {product_name}

ОПИСАНИЕ ТОВАРА:
- Название: {product_name}
- Описание: {product_description}
- Старая цена: {old_price} BYN
- Новая цена: {new_price} BYN
- Скидка: {discount_percent}%

ФОТО ТОВАРА:
{product_images} - массив URL фотографий

ПРЕИМУЩЕСТВА:
{benefits} - массив преимуществ с иконками

ХАРАКТЕРИСТИКИ:
{characteristics} - массив характеристик

ОТЗЫВЫ:
{reviews} - массив отзывов с фото, текстом, именем

ДОСТАВКА И ОПЛАТА:
- Доставка: {delivery_info}
- Оплата: {payment_info}
- Гарантия: {warranty_info}

СТРУКТУРА:
1. Hero секция с фото товара, ценой, формой заказа и бейджем скидки
2. Галерея фотографий товара (карусель с центрированием крайних фото)
3. Детальное описание с преимуществами
4. Блок ключевых преимуществ с иконками
5. Карусель отзывов (1 отзыв видно, фото крупное)
6. Финальная форма заказа с таймером обратного отсчета (обнуляется в 00:00)

ТЕХНИЧЕСКИЕ ТРЕБОВАНИЯ:
- Таймер обратного отсчета на сутки с обнулением в полночь
- Карусели галереи и отзывов с правильным центрированием
- Форматирование телефона +375 (__) ___-__-__
- Отправка заказа в Telegram в формате: "Новый заказ: {product_name}\nИмя: {name}\nНомер телефона: {phone}\nСтоимость: {price}\nВремя заказа: {time}"
- TikTok Pixel с ID: {tiktok_pixel_id}
```

```

{variants} - объект с вариантами (цвет, размер, комплектация)

Пример структуры вариантов:
{
  "color": ["Белый", "Черный", "Серый"],
  "size": ["S", "M", "L", "XL"],
  "package": ["Базовый", "Стандарт", "Премиум"]
}

ЦЕНЫ:
- Базовая цена: {base_price} BYN
- Цены по вариантам: {variant_prices} - объект

{product_images} - объект с фото для каждого варианта

ТАБЛИЦА РАЗМЕРОВ:
{size_guide} - таблица с размерами (если применимо)

СТРУКТУРА:
1. Hero с селектором вариантов (цвет, размер, комплектация)
2. Динамическое изменение цены при выборе варианта
3. Галерея с фильтрацией по вариантам
4. Модальное окно с таблицей размеров
5. Остальные блоки как в стандартном лендинге

ОСОБЕННОСТИ:
- При выборе варианта автоматически меняется цена и фото
- Цена рассчитывается динамически на основе выбранных вариантов
- В форме заказа передаются выбранные варианты
```

## Универсальные компоненты

### Таймер обратного отсчета

```
Добавь таймер обратного отсчета:
- Формат: ЧЧ:ММ:СС
- Обнуление в 00:00 ежедневно
- Автоматический пересчет до следующей полночи
- Размещение в hero и финальной CTA секциях
```

### Карусель галереи

```
Добавь карусель фотографий:
- Адаптивная: на мобильных горизонтальный скролл, на десктопе grid
- Центрирование крайних фото при прокрутке
- Навигация стрелками на мобильных
- Точки индикации текущего слайда
- Плавная прокрутка
```

### Карусель отзывов

```
Добавь карусель отзывов:
- 1 отзыв видно на экране
- Крупное фото отзыва
- Текст не растягивается, ограничен по высоте
- Навигация стрелками
- Точки индикации
```

### Форма заказа

```
Создай форму заказа с полями:
- Имя (обязательное)
- Телефон (обязательное, форматирование +375)
- Email (опционально)
- Комментарий (опционально)

Валидация:
- Проверка заполненности обязательных полей
- Форматирование телефона
- Отправка в Telegram через sendCPA.php
```

- [ ] Телефон форматируется корректно
- [ ] Telegram интеграция работает
- [ ] TikTok Pixel подключен
- [ ] Все изображения оптимизированы
- [ ] Анимации плавные
- [ ] CTA кнопки заметные
- [ ] Социальные доказательства видны
- [ ] Цены выделены визуально
