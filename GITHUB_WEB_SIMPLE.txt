ПРОСТАЯ ИНСТРУКЦИЯ: ЗАГРУЗКА В GITHUB ЧЕРЕЗ ВЕБ-ИНТЕРФЕЙС
═══════════════════════════════════════════════════════════════

САМЫЙ ПРОСТОЙ СПОСОБ! ⭐
───────────────────────────────────────────────────────────────

Не нужны: GitHub Desktop, командная строка, настройка DNS


ШАГ 1: РЕГИСТРАЦИЯ НА GITHUB
───────────────────────────────────────────────────────────────

1. Зайдите на github.com
2. Нажмите "Sign up"
3. Заполните форму (username, email, password)
4. Подтвердите email


ШАГ 2: СОЗДАНИЕ РЕПОЗИТОРИЯ
───────────────────────────────────────────────────────────────

1. Нажмите "+" (вверху справа) → "New repository"
2. Имя: telegram-landing-bot
3. Выберите "Private" (приватный)
4. НЕ ставьте галочки на README, .gitignore, license
5. Нажмите "Create repository"


ШАГ 3: ЗАГРУЗКА ФАЙЛОВ
───────────────────────────────────────────────────────────────

СПОСОБ 1: DRAG & DROP (САМЫЙ ПРОСТОЙ) ⭐

1. На странице репозитория нажмите "uploading an existing file"
2. Откройте папку проекта: D:\Сайты\creativelandingbotfixed
3. Выберите все файлы (Ctrl + A)
4. Перетащите файлы в окно браузера
5. Внизу введите: "Initial commit"
6. Нажмите "Commit changes"
7. Готово!


ВАЖНО: ЧТО НЕ ЗАГРУЖАТЬ
───────────────────────────────────────────────────────────────

❌ НЕ загружайте:
   - .env (ваши ключи!)
   - *.db (база данных)
   - *.log (логи)
   - generated_landings/ (сгенерированные файлы)
   - venv/ (виртуальное окружение)

✅ Загружайте:
   - Все .py файлы
   - requirements.txt
   - Procfile
   - .gitignore
   - env.example
   - Папки: backend/, examples/
   - Все .md файлы


ШАГ 4: ПРОВЕРКА
───────────────────────────────────────────────────────────────

1. Обновите страницу (F5)
2. Должны увидеть все файлы
3. Проверьте, что .env НЕ виден (хорошо!)


ОБНОВЛЕНИЕ ПРОЕКТА (когда вносите изменения)
───────────────────────────────────────────────────────────────

1. Найдите файл в репозитории
2. Нажмите на файл
3. Нажмите "Edit" (карандаш)
4. Внесите изменения
5. Введите описание изменений
6. Нажмите "Commit changes"


ПРЕИМУЩЕСТВА ВЕБ-ВЕРСИИ
───────────────────────────────────────────────────────────────

✅ Не нужно устанавливать программы
✅ Всё через браузер
✅ Не зависит от DNS/прокси
✅ Работает с любого компьютера
✅ Простой интерфейс


ГОТОВО!
───────────────────────────────────────────────────────────────

После загрузки проект готов к деплою на Railway!
