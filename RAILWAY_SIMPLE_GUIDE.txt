ПРОСТАЯ ИНСТРУКЦИЯ: ЗАПУСК БОТА НА RAILWAY
═══════════════════════════════════════════════════════════════

Для не-программистов - всё максимально просто!


ШАГ 1: РЕГИСТРАЦИЯ
═══════════════════════════════════════════════════════════════

1. Откройте сайт: railway.app
2. Нажмите "Start a New Project"
3. Войдите через GitHub (самый простой способ)
   - Нажмите "Login with GitHub"
   - Разрешите доступ
   - Готово!


ШАГ 2: ПОДКЛЮЧЕНИЕ ПРОЕКТА
═══════════════════════════════════════════════════════════════

1. Нажмите "New Project" (зеленая кнопка)
2. Выберите "Deploy from GitHub repo"
3. Выберите ваш репозиторий с ботом
4. Нажмите "Deploy Now"
5. Railway начнет автоматическую настройку


ШАГ 3: ДОБАВЛЕНИЕ БАЗЫ ДАННЫХ
═══════════════════════════════════════════════════════════════

1. В проекте нажмите "New" (кнопка справа)
2. Выберите "Database"
3. Выберите "Add PostgreSQL"
4. Готово! База данных создана автоматически


ШАГ 4: УСТАНОВКА ПЕРЕМЕННЫХ ОКРУЖЕНИЯ
═══════════════════════════════════════════════════════════════

1. В проекте найдите карточку с вашим ботом
2. Нажмите на неё
3. Перейдите во вкладку "Variables"
4. Нажмите "New Variable" для каждой переменной:

   Переменная 1:
   Key:   TELEGRAM_BOT_TOKEN
   Value: ваш_токен_от_BotFather

   Переменная 2:
   Key:   LLM_PROVIDER
   Value: openai

   Переменная 3:
   Key:   OPENAI_API_KEY
   Value: ваш_ключ_от_OpenAI

   Переменная 4:
   Key:   LLM_MODEL
   Value: gpt-4o

   Переменная 5:
   Key:   DATABASE_URL
   Value: (уже добавлен автоматически, проверьте что есть)

5. После добавления каждой переменной нажмите "Add"


ШАГ 5: ПРОВЕРКА ЗАПУСКА
═══════════════════════════════════════════════════════════════

1. В карточке бота перейдите во вкладку "Deployments"
2. Выберите последний деплой
3. Перейдите во вкладку "Logs"
4. Должны увидеть:
   ✓ Конфигурация валидна
   ✓ База данных инициализирована
   ✓ Бот создан
   Бот запущен и готов к работе

5. Если видите ошибки - проверьте переменные окружения


ШАГ 6: ТЕСТИРОВАНИЕ
═══════════════════════════════════════════════════════════════

1. Откройте Telegram
2. Найдите вашего бота
3. Отправьте команду /start
4. Бот должен ответить!


ЧТО ДЕЛАТЬ ЕСЛИ НЕ РАБОТАЕТ
═══════════════════════════════════════════════════════════════

1. Проверьте логи (шаг 5)
2. Проверьте все переменные окружения
3. Убедитесь что:
   - TELEGRAM_BOT_TOKEN правильный
   - OPENAI_API_KEY правильный
   - DATABASE_URL есть в списке переменных

4. Если всё правильно, но не работает:
   - Нажмите "Settings" → "Restart" (перезапуск)
   - Или напишите в поддержку Railway


ВАЖНЫЕ МОМЕНТЫ
═══════════════════════════════════════════════════════════════

✓ Всё делается через веб-интерфейс (не нужны команды)
✓ Railway автоматически обновляет бота при изменениях в GitHub
✓ Логи можно смотреть в реальном времени
✓ База данных настраивается автоматически


ГОТОВО!
═══════════════════════════════════════════════════════════════

После выполнения всех шагов бот будет работать 24/7 на Railway!
