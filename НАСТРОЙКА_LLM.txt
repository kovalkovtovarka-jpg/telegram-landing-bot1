1. УСТАНОВИТЕ ЗАВИСИМОСТИ:
   
   pip install -r requirements.txt

2. НАСТРОЙТЕ .env ФАЙЛ:

   Для OpenAI (рекомендуется):
   ────────────────────────────
   LLM_PROVIDER=openai
   OPENAI_API_KEY=sk-ваш-ключ-здесь
   LLM_MODEL=gpt-4o
   LLM_TEMPERATURE=0.3
   LLM_MAX_TOKENS=8000

   Для Anthropic Claude:
   ────────────────────────────
   LLM_PROVIDER=anthropic
   ANTHROPIC_API_KEY=sk-ant-ваш-ключ-здесь
   LLM_MODEL=claude-3-5-sonnet-20241022
   LLM_TEMPERATURE=0.3
   LLM_MAX_TOKENS=8000

   Для Google Gemini:
   ────────────────────────────
   LLM_PROVIDER=google
   GOOGLE_API_KEY=ваш-ключ-здесь
   LLM_MODEL=gemini-1.5-pro
   LLM_TEMPERATURE=0.3
   LLM_MAX_TOKENS=8000

Google Gemini:

═══════════════════════════════════════════════════════════════
  РЕШЕНИЕ ПРОБЛЕМ:
═══════════════════════════════════════════════════════════════

Проблема: "API ключ не установлен"
  → Проверьте .env файл
  → Убедитесь, что ключ правильный
  → Нет лишних пробелов

Проблема: "Неполный код"
  → Увеличьте LLM_MAX_TOKENS до 10000
  → Используйте gpt-4o или claude-3-5-sonnet
  → Снизьте температуру до 0.2

Проблема: "Плохое качество"
  → Используйте более новую модель
  → Проверьте, что все данные в промпте
  → Температура должна быть 0.2-0.3
