📍 ШАГ 1: ПОЛУЧИТЕ TELEGRAM BOT TOKEN
─────────────────────────────────────────────────────────────────
1. Откройте Telegram и найдите @BotFather
2. Отправьте команду: /newbot
3. Введите имя бота (например: Landing Generator)
4. Введите username бота (например: landing_gen_bot)
5. Скопируйте полученный токен (формат: 1234567890:ABCdefGHI...)

📍 ШАГ 2: ПОЛУЧИТЕ OPENAI API KEY
─────────────────────────────────────────────────────────────────
1. Перейдите на: https://platform.openai.com/api-keys
2. Войдите в аккаунт или зарегистрируйтесь
3. Нажмите "Create new secret key"
4. Скопируйте ключ (показывается только один раз!)
5. ВАЖНО: Пополните баланс ($5-10 минимум)

📍 ШАГ 3: СОЗДАЙТЕ ФАЙЛ .env
─────────────────────────────────────────────────────────────────
Выполните в терминале:

cd /home/ubuntu/creativelandingbot/creativelandingbot/
cp .env.example .env
nano .env

Заполните обязательные параметры:
- TELEGRAM_BOT_TOKEN=ваш_токен_от_botfather
- OPENAI_API_KEY=ваш_ключ_от_openai

Сохраните файл (Ctrl+O, Enter, Ctrl+X)

📍 ШАГ 4: УСТАНОВИТЕ ЗАВИСИМОСТИ
─────────────────────────────────────────────────────────────────
cd /home/ubuntu/creativelandingbot/creativelandingbot/
pip install -r requirements.txt

📍 ШАГ 5: СОЗДАЙТЕ ДИРЕКТОРИЮ ДЛЯ ФАЙЛОВ
─────────────────────────────────────────────────────────────────
mkdir -p generated_landings
chmod 755 generated_landings

📍 ШАГ 6: ПРОВЕРЬТЕ КОНФИГУРАЦИЮ
─────────────────────────────────────────────────────────────────
python3 -c "from backend.config import Config; Config.validate(); print('✓ OK')"

Если видите "✓ OK" - можно запускать!
Если ошибка - проверьте .env файл

📍 ШАГ 7: ЗАПУСТИТЕ БОТА
─────────────────────────────────────────────────────────────────
# Вариант 1: Обычный запуск (с выводом в консоль)
python3 main.py

# Вариант 2: Запуск в фоне
nohup python3 main.py > bot.log 2>&1 &

# Просмотр логов (если запущен в фоне)
tail -f bot.log

📍 ШАГ 8: ПРОТЕСТИРУЙТЕ БОТА
─────────────────────────────────────────────────────────────────
1. Откройте Telegram
2. Найдите вашего бота по username
3. Отправьте: /start
4. Бот должен ответить приветствием!

🎯 БЫСТРЫЙ ТЕСТ ГЕНЕРАЦИИ
─────────────────────────────────────────────────────────────────
Отправьте боту:
"Создай лендинг для подушки, цена 99 BYN"

Бот должен начать генерацию (займет 30-60 секунд)

🆘 ЕСЛИ ЧТО-ТО НЕ РАБОТАЕТ
─────────────────────────────────────────────────────────────────
1. Проверьте bot.log:
   tail -f bot.log

2. Проверьте, что бот запущен:
   ps aux | grep main.py

3. Проверьте баланс OpenAI:
   https://platform.openai.com/account/billing

4. Читайте полный отчет:
   /home/ubuntu/bot_analysis_report.md
